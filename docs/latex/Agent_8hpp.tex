This file contains the definition of the Agent class. The Agent class represents an agent in the network. It is templated by the type of the agent\textquotesingle{}s id and the size of agents, which must both be unsigned integrals. It is also templated by the Delay type, which must be a numeric (see utility/\+Type\+Traits/is\+\_\+numeric.\+hpp) and represents the spatial or temporal (depending on the type of the template) distance between the agent and the one in front of it.
